# motet
Learn vocabularies

## Installieren auf dem Android-Smartphone

Über den Playstore das Programm termux installieren. Das ist eine Linux-Simulation.

Darin über

pkg install python

Python installieren.

Dann mit

termux-setup-storage

Zugriff auf die Smartphone-Verzeichnisse geben.

Dann die Dateien motet.py und vocabulary.py nach Android/data/com.termux/files schieben.

Dann kann man es in termux unter ~/storage/external-0/ finden und ausführen.
